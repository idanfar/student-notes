## 🚀 כיצד להצטרף?

אנחנו שמחים לקבל תורמים חדשים בכל שלב! כך תוכלו להתחיל:

1. **עיינו בסוגיות הפתוחות** — בקרו ב-[Issues](https://github.com/idanfar/student-notes/issues) וחפשו תגיות כמו `good first issue` או `help wanted`.

2. **עשו Fork לפרויקט** — לחצו על כפתור ה-Fork בפינה הימנית העליונה של הדף.

3. **צרו Branch חדש** — קראו לו שם תיאורי, לדוגמה:
   ```bash
   git checkout -b feature/add-linear-algebra-notes
   ```

4. **הוסיפו את התרומה שלכם** — בין אם מדובר בהערות לקורס, תיקון שגיאה, או שיפור תיעוד.

5. **שלחו Pull Request** — תארו בקצרה מה שינויתם ומדוע. נשמח לבדוק ולמזג!

> 💡 **טיפ:** קראו את [CONTRIBUTING.md](./CONTRIBUTING.md) לפני שמתחילים — תמצאו שם קונבנציות כתיבה ודרישות הגשה.
